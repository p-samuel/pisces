
**1. Basic Demo**

**2. Carousel Demo** 
- Showcases scrollable content layouts using ScrollView and nested containers. 
- Demonstrates how to build carousel-like layouts with multiple sections and content that can be scrolled vertically.

**3. Gradients Demo**
- Displays three gradient types—linear, radial, and sweep—with interactive color theme switching. Users tap to cycle through five themes.

**4. Lifecycles Demo**
- Comprehensive example showing both activity and view lifecycle events with visual feedback. 
- Demonstrates pre/post creation hooks, resume/pause states, and includes a "ninja fruit" animation effect.

**5. Navigation Demo**
- Implements a screen management and multi-screen navigation using the ScreenManager API.
- Demonstrate screen transitions, and inter-screen communication patterns.

**6. Santorini Demo**
- Features a scrollable content screen with an image, title, and formatted text description. 
- Demonstrates image loading, text justification, swipe detection, and elevation effects—great.

**7. Gestures Demo** 
- Interactive touch and swipe detection example that displays coordinates and swipe directions in real-time. 
- Users swipe in different directions to see coordinate updates and swipe detection.
